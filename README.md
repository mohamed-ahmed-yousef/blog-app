# Slash Frontend Internship task

## Getting Started

1. Step one

```sh
    git clone https://github.com/mohamed-ahmed-yousef/blog-app.git
    cd blog-app
    pnpm install
    pnpm dev
```

1. Step two

- add the `NEXT_PUBLIC_BACKEND_URL` in `.env` file
